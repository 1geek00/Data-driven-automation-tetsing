# Playwright Python Data-Driven Testing

A simple automation project demonstrating **Data-Driven Testing (DDT)** using **Playwright, Python, Pytest, and CSV**.

## Tech Stack

- Python
- Playwright
- Pytest
- CSV

## Features

- Automated login test
- Reads test data from a CSV file
- Simple and beginner-friendly project structure

## Project Structure

```
├── tests/
│   └── login.py
├── data.csv
└── README.md
```

## Run the Project

Install dependencies:

```bash
pip install -r requirements.txt
playwright install
```

Run the test:

```bash
pytest tests/login.py --headed
```

## Learning Outcomes

- Playwright with Python
- Data-Driven Testing
- Pytest
- CSV file handling
- Browser automation
